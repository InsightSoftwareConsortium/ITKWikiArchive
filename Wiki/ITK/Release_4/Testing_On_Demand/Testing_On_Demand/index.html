Goal

The purpose of this effort is to make possible to have

  • A collection of client machines that donate time for testing
  • A central scheduler of testing tasks hosted at CDash
  • A process for requesting a version X of code Y to be tested in a platform Z

Implementation

  • This has been implemented as part of CDash
