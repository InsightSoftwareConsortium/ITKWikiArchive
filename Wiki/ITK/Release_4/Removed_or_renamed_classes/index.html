The following table lists each ITKv3 class that was removed or renamed.

  • Many of the classes, e.g. the Functor classes, were renamed

when then were moved from Review into other modules.

  • The itkOpt classes, replaced their non-optimized counterparts when they were

moved from Review.

  • For removed or renamed classes that existed outside of Review,

migration documents are linked.

  • All classes that contained patented code were removed.
