The ITKv4 release candidate 3 has been tagged, and is available for testing. To obtain the source code, 

git clone http://itk.org/ITK.git
cd ITK
git checkout -q --detach v4.0rc03

For more details, please see 

http://www.itk.org/ITKWikiArchive/Wiki/ITK/Git


  • Important note:* Please do not expect backward compatibility with

these release candidates. The API may still undergo changes as improvements are made.

Release candidates will be tagged approximately every week, provided that the software quality dashboard is in decent shape 

http://public.kitware.com/dashboard.php?name=itk


Testing and feedback are encouraged. Impressions or patches will help improve the result as refinement and bug-fixes are made for the final release.
